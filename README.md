# 🔭 History Heroes

A kid-friendly biography explorer that shows famous people born on today's date, powered by the Wikipedia API.

## Features
- 🎂 Daily rotating birthday heroes (live from Wikipedia "On This Day" API)
- 🔍 Search any famous person by name — handles nicknames & punctuation (e.g. J.K. Rowling)
- 💡 Real famous quotes ticker powered by Quotable API
- 📖 Bio popup with photo, description, and Wikipedia link
- 🌌 Space-themed design with kid-friendly fonts
- 📱 Responsive layout

## Tech Stack
- Vite + React + JavaScript
- Wikipedia REST API (no key required)
- Quotable API (no key required)
- Deployed on Vercel

## Run Locally
npm install
npm run dev
