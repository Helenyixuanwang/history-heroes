# 🔭 History Heroes

A kid-friendly biography explorer that shows famous people born on today's date, powered by the Wikipedia API.

## Features
- 🎂 Daily rotating famous birthdays (live from Wikipedia)
- 🔍 Search any famous person by name
- 📖 Bio popup with photo, description, and Wikipedia link
- 🌌 Space-themed design, kid-friendly fonts

## Tech Stack
- React + Vite
- Wikipedia REST API (no API key needed)

## Run Locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or connect your GitHub repo at https://vercel.com — it auto-detects Vite.

## Deploy to Netlify

```bash
npm run build
# drag-and-drop the dist/ folder at https://app.netlify.com/drop
```

Or connect your GitHub repo at https://netlify.com.
